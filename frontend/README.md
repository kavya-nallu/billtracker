# 💸 Bill Tracker (MERN Stack)

A streamlined financial management tool designed to help users monitor spending, manage recurring bills, and gain insights into their monthly expenses.

## ✨ Key Features
* **Secure Auth:** User login and registration using JWT.
* **Expense Management:** Full CRUD operations to add, view, and delete bills.
* **Categorization:** Organize spending by categories like Food, Rent, and Utilities.
* **Responsive Design:** Works perfectly on mobile and desktop.

## 🛠️ Tech Stack
* **Frontend:** React.js, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
